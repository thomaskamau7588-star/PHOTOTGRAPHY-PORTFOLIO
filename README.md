# Photography Website

## Description

This is a simple photography website created using HTML and CSS. It showcases my favorite photographs, includes a short biography, and provides a contact form for visitors.

## Features

* Home section with an introduction
* Photo gallery
* Contact form
* Custom CSS styling

## Technologies Used

* HTML5
* CSS3
* Github pages
* Git
* Github

## Setup Instructions

1. Clone or download this repository.
```bash
git clone https://github.com/thomaskamau7588-star/photography-website.git
```
2. Open the project folder.
3. Open `index.html` in your web browser.
